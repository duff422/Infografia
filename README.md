# Infografía: Radiografía Ósea

Esta infografía explica la física de la radiación aplicada a la radiografía ósea, cubriendo aspectos desde la generación de rayos X hasta las aplicaciones biomédicas y riesgos asociados.

## Estructura del Proyecto

- `index.html`: Archivo principal de la infografía.
- `css/styles.css`: Estilos CSS para el diseño y layout.

## Tecnologías Utilizadas

- HTML5
- CSS3 (con variables CSS y Grid)
- Fuentes de Google Fonts: Orbitron, Rajdhani, Share Tech Mono
- SVG para ilustraciones

## Visualización

Abre `index.html` en un navegador web para ver la infografía completa.

## Contenido

La infografía cubre:
1. Tipo de radiación utilizada (Rayos X)
2. Esquema del proceso de radiografía
3. Mecanismos de interacción (efecto fotoeléctrico y Compton)
4. Aplicación biomédica en diagnóstico óseo
5. Riesgos biológicos y ventajas
6. Conclusión con síntesis física

Parte B de una serie sobre Física de la Radiación.
